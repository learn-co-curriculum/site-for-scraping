Android for Developers

Android for Developers will give you the skills you need to build an Android app.

In this part-time course, students with some programming experience will become proficient Android Developers. Over the course of ten weeks, students will use Java and industry standard tools and services to build and deploy an Android app to the Google Play Store.

Familiarity with Java is a huge plus, but to enroll, all students should have at least one year of object-oriented programming experience, including Objective-C, Python, Ruby, C#, JavaScript, or C++. Tuition is $3,500.

Curriculum

We teach the Android skills employers value.

Request a Syllabus

Fundamentals

Use your existing programming knowledge to get up to speed with Java. Then dive into Google API's, using Android Studio to develop, debug and measure your applications. Finally, deploy your apps to hundreds of millions of users in Google Play.

User Interface

Learn Android's powerful XML layout system. Use fragments to decouple UI components from underlying activities, and allow your interfaces to easily scale from wearables to televisions. Use the resource system to quickly modify your applications assets without writing a single line of code.

Data and APIs

Learn low-level Java and Android HTTPRequest Libraries—and how to keep your internet requests performant and off of the main UI thread. Persist your data by saving to local JSON or to database using SQLite. Finally, use Square's Retrofit library to abstract your REST Client and clean up your network code.

Services

Synchronize your app data and handle notifications with Google Cloud Messaging and other Google Service APIs. Use Intents and Content Providers to leverage Android's ability to share data and services between applications.
